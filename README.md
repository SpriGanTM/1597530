# Random Quote Generator

## Автор
Тихоньких Макар Дмитриевчи 454 группа

## Описание
Приложение с графическим интерфейсом для генерации случайных цитат.

Функционал:
- генерация случайной цитаты
- фильтрация по автору и теме
- добавление новых цитат
- сохранение истории в JSON

## Использование
1. Запустить файл main.py
2. Нажать кнопку "Сгенерировать цитату"
3. При необходимости ввести фильтр (автор или тема)
4. Можно добавить свою цитату через поля ввода

## Пример проверки
- Пустые поля → ошибка
- Нет совпадений → предупреждение
- История сохраняется после перезапуска
